# 岗位信息整理标准

## 必要字段

| 字段 | 说明 | 示例 |
|------|------|------|
| 职位名称 | 原始职位标题 | 高级Java开发工程师 |
| 公司名称 | 全称，避免简称 | 北京字节跳动科技有限公司 |
| 工作地点 | 城市 + 区（如有） | 北京市海淀区 |
| 薪资范围 | 原始描述 + 归一化 | 25-40K·16薪 / 25-40K |
| 来源URL | 岗位详情页链接 | https://... |

## 推荐字段

| 字段 | 说明 | 示例 |
|------|------|------|
| 发布时间 | 原始时间描述 | 3天前发布 |
| 工作年限 | 经验要求 | 3-5年 |
| 学历要求 | 最低学历 | 本科 |
| 职位描述 | JD摘要（可选） | 负责后端服务开发... |
| 技能标签 | 关键技能提取 | Java, Spring, MySQL |
| 公司规模 | 规模信息 | 10000人以上 |
| 行业领域 | 公司行业 | 互联网/移动互联网 |

## 去重规则

### 归一化键生成

使用「公司名 + 职位名 + 地点」组合：

```
归一化键 = normalize(公司名) + "|" + normalize(职位名) + "|" + normalize(地点)
```

### 归一化规则

1. **公司名**：去除「有限公司」「科技」「网络」等通用后缀
2. **职位名**：去除「急聘」「高薪」「诚聘」等修饰词
3. **地点**：统一为「城市」级别，去除区县

### 示例

| 原始 | 归一化键 |
|------|----------|
| 字节跳动-高级Java工程师-北京 | 字节跳动\|高级java工程师\|北京 |
| 北京字节跳动科技-急聘Java高级开发-海淀区 | 字节跳动\|java高级开发\|北京 |

## 数据质量检查

- [ ] 公司名称完整（非简称）
- [ ] 薪资格式统一
- [ ] URL 可访问
- [ ] 无重复记录
- [ ] 时间信息有效
