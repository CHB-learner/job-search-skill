# 多平台搜索 URL 模板

本文档提供四大招聘平台的搜索 URL 构造指南，方便快速访问和检索岗位。

---

## 1. BOSS直聘

### 搜索 URL 模板

```
https://www.zhipin.com/web/geek/job?query={关键词}&city={城市代码}
```

### 常用城市代码

| 城市 | 代码 |
|------|------|
| 北京 | 101010100 |
| 上海 | 101020100 |
| 广州 | 101280100 |
| 深圳 | 101280600 |
| 杭州 | 101210100 |
| 成都 | 101270100 |
| 南京 | 101190100 |
| 武汉 | 101200100 |
| 西安 | 101110100 |
| 苏州 | 101190400 |

### 筛选参数

| 参数 | 说明 | 示例 |
|------|------|------|
| `query` | 搜索关键词 | Java后端 |
| `city` | 城市代码 | 101010100 |
| `experience` | 经验要求 | 103-5年 |
| `salary` | 薪资范围 | 405-410 |

### 示例 URL

```
# 北京 Java 后端
https://www.zhipin.com/web/geek/job?query=Java后端&city=101010100

# 上海 前端开发 3-5年经验
https://www.zhipin.com/web/geek/job?query=前端开发&city=101020100&experience=103
```

### 注意事项

- 需要登录才能查看完整信息
- 反爬机制严格，不建议自动化批量访问
- 可能触发安全验证（滑块、验证码）

## 3. 前程无忧 (51job)

### 搜索 URL 模板

```
https://search.51job.com/list/{城市代码},000000,0000,00,9,99,{关键词},2,1.html
```

### 常用城市代码

| 城市 | 代码 |
|------|------|
| 北京 | 010000 |
| 上海 | 020000 |
| 广州 | 030200 |
| 深圳 | 040000 |
| 杭州 | 080200 |
| 成都 | 090200 |
| 南京 | 070200 |
| 武汉 | 180200 |
| 西安 | 200200 |
| 苏州 | 070300 |

### URL 参数说明

URL 格式：`list/{城市},{地区},{行业},{职能},{工作年限},{学历},{关键词},{排序},{页码}.html`

| 位置 | 说明 | 示例值 |
|------|------|--------|
| 城市代码 | 目标城市 | 010000 |
| 地区代码 | 区县筛选 | 000000 (不限) |
| 行业代码 | 行业筛选 | 0000 (不限) |
| 职能代码 | 职能筛选 | 00 (不限) |
| 工作年限 | 经验要求 | 9 (不限) |
| 学历要求 | 学历筛选 | 99 (不限) |
| 关键词 | 搜索词 | Java |
| 排序方式 | 结果排序 | 2 (默认) |
| 页码 | 分页 | 1 |

### 工作年限代码

| 代码 | 含义 |
|------|------|
| 1 | 无经验/应届生 |
| 2 | 1年以下 |
| 3 | 1-3年 |
| 4 | 3-5年 |
| 5 | 5-10年 |
| 6 | 10年以上 |
| 9 | 不限 |

### 示例 URL

```
# 北京 Java
https://search.51job.com/list/010000,000000,0000,00,9,99,Java,2,1.html

# 上海 前端 3-5年
https://search.51job.com/list/020000,000000,0000,00,4,99,前端,2,1.html
```

## 检索建议

### 并行检索策略

1. 打开 4 个浏览器标签页
2. 分别访问 BOSS、猎聘、前程无忧、智联
3. 使用相同关键词搜索
4. 记录各平台结果，后续去重

### 时间过滤建议

- 优先：近 3 天
- 次选：近 7 天
- 补充：近 14 天

### 关键词组合建议

```
{岗位名} + {核心技能}

示例：
- Java后端 Spring
- 前端开发 Vue React
- Python开发 Django
- 产品经理 B端
```
