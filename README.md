

# 🔥 JobSearch Skill

**一键打通全网招聘信息的找工作 Skill**

厌倦在多个招聘网站来回切换？
**JobSearch Web** 帮你一次搜全网岗位，自动整理、去重、筛选，省时又高效。

---

## 🚀 3秒上手

告诉我你的目标：

> 岗位 / 城市 / 薪资 / 是否远程

我立刻帮你在：

* BOSS直聘
* 猎聘
* 51job（前程无忧）
* 智联招聘

同步检索，并整理成清晰结构化列表。

---

## 🧠 越搜越精准

支持多条件过滤与偏好调整，逐步逼近你真正想要的岗位组合。
不再被“海投式搜索”淹没。

---

## ✨ 核心功能

* 🔎 **多平台聚合**：主流招聘网站一网打尽
* 🧹 **智能去重**：自动识别重复岗位（公司 + 职位 + 地点）
* 📊 **标准化输出**：统一格式表格，直接可投递
* 🧭 **精细筛选**：城市 / 薪资 / 经验 / 行业多维过滤
* 🔗 **原始链接保留**：一键跳转官方岗位页
* ⚡ **快速更新**：优先抓取近 7 天新岗位

---

## ⚙️ 技术亮点

* 多源并行检索策略，效率优先
* 归一化去重逻辑（稳定 URL + 语义匹配）
* 可扩展自动化方案（Playwright / Spring Boot 参考实现）
* 结构化输出模板，方便后续分析与投递
